# Drone Telemetry Server & Dashboard (V1.0)

## Prerequisites

- **Node.js** `v18+`
- **npm** `v9+`
- **Git**

---

## How to Build and Run (Development)

### 1. Clone the Repository

### 2. npm install

### 3. to run the app
  ## cd backend 
  ## npm run start:dev
  ## cd frontend
  ## npm run dev

### 4. for build :
  ## cd backend
  ## npm run build
  ## cd frontend
  ## npm run build
